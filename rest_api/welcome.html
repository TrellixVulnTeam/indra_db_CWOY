The INDRA Database currently integrates the following natural language processing systems at scale:
  • REACH - for biology
  • Sparser - for biology
Readers coming soon:
  • RLIMS-P
  • TRIPS
  • ISI Reader(see on github)
With content drawn from:
  • PubMed - ~28 million abstracts Fall 2018
  • PubMed Central - ~5 million fulltext Fall 2018
  • Elsevier - ~0.7 million fulltext (requires special access) Fall 2018
We also collect information from these databases:
  • Pathway Commons database
  • BEL Large Corpus
  • biogrid
  • biopax
  • SIGNOR
  • Tas
These databases are retrieved using the tools in indra.sources. The statements extracted from all of these sources are stored and updated in the database.

This work was funded by ARO grant W911NF‐15‐1‐0544 under the DARPA Communicating with Computers program.
